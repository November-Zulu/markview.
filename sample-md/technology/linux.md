# Linux

Linux is a family of open-source, Unix-like operating systems based on the **Linux kernel**, first released by *Linus Torvalds* on September 17, 1991. Combined with the GNU project's userland tools and libraries, Linux has grown into one of the most important software projects in computing history -- powering everything from smartphones to supercomputers.

## History

### Origins

Linus Torvalds created the Linux kernel in 1991 while attending the **University of Helsinki** in Finland. Frustrated by the licensing restrictions of Minix (a Unix-like teaching operating system), Torvalds set out to build a Unix-compatible kernel that anyone could use and modify freely.

The GNU Project, started by **Richard Stallman** in 1983, had already created many Unix-compatible components -- compilers, editors, shells, and utilities -- but lacked a functional kernel. Torvalds' creation filled this gap, and GNU components were integrated with the Linux kernel to form complete operating systems.

### The Name

Torvalds originally wanted to call his project **"Freax"** (a portmanteau of "free," "freak," and "x" for Unix) but didn't consider "Linux" suitable. According to historical accounts, *Ari Lemmke*, Torvalds' coworker at Helsinki University of Technology, named the project "Linux" on the FTP server without consulting Torvalds. The name stuck.

### The GNU/Linux Debate

The Free Software Foundation advocates calling the system **"GNU/Linux"** to emphasize the GNU software's importance in complete distributions. However, most media outlets and major distributions simply use "Linux." This naming debate remains a point of discussion in the open-source community.

## Licensing

The Linux kernel is released under the **GNU General Public License version 2** (GPLv2). This copyleft license requires that anyone distributing software based on the code must:

- Make source code modifications available
- Distribute under identical license terms
- Preserve copyright notices

Torvalds has stated the kernel will *permanently* remain under GPLv2, specifically objecting to GPLv3's digital rights management provisions.

## Major Distributions

A Linux **distribution** (or "distro") combines the Linux kernel with system software and application libraries to create a complete operating system. Popular distributions include:

### Community-Driven

| Distribution | Based On | Known For |
|-------------|----------|-----------|
| **Debian** | Independent | Stability, massive package repository |
| **Ubuntu** | Debian | User-friendliness, desktop popularity |
| **Fedora** | Independent | Cutting-edge features, Red Hat upstream |
| **Arch Linux** | Independent | Minimalism, rolling releases, user control |
| **Linux Mint** | Ubuntu | Familiar desktop experience, ease of use |
| **openSUSE** | Independent | YaST configuration tool, enterprise roots |

### Commercial

- **Red Hat Enterprise Linux (RHEL)** -- Industry-standard enterprise distribution
- **SUSE Linux Enterprise** -- European enterprise favorite
- **ChromeOS** -- Google's Linux-based OS for Chromebooks

### Specialized

- **Kali Linux** -- Penetration testing and security auditing
- **Red Star OS** -- North Korea's national operating system
- **SteamOS** -- Valve's gaming-focused distribution

## Usage and Market Presence

### Servers and Infrastructure

Linux **dominates** the server market:

- Powers over **96%** of the top one million web servers
- Runs on all **500** of the world's fastest supercomputers
- Accounts for approximately **80%** of public cloud workloads across AWS, Azure, and Google Cloud Platform

### Mobile Devices

**Android**, based on the Linux kernel, held approximately 68.61% of mobile device market share as of April 2023. Combined with traditional Linux, the kernel powers the **largest installed base** of any general-purpose operating system.

### Desktop

Linux comprises roughly **4.3%** of desktop computer usage as of October 2024, behind:

1. Windows -- 73.4%
2. macOS -- 15.5%
3. Linux -- 4.3%

While desktop share remains modest, it has been steadily growing, especially among developers and technical professionals.

### Film and Entertainment

Linux has become the **film industry standard**. Major studios including *DreamWorks Animation*, *Pixar*, *Weta Digital*, and *Industrial Light & Magic* have migrated their production pipelines to Linux platforms.

## Technical Architecture

Linux-based systems employ a **monolithic kernel** that handles:

- Process control and scheduling
- Memory management
- Networking stack
- Peripheral and device access
- File systems

Device drivers integrate directly into the kernel or load as **modules** during runtime. Most distributions include the GNU C Library (glibc), approximately 380 POSIX-compatible system calls, and standard Unix tools.

### Display Systems

Modern Linux systems are increasingly transitioning from the **X11** (X Window System) display server toward **Wayland**, which offers:

- Better security (application isolation)
- Simpler architecture
- Improved performance for compositing

## Development Model

> Linux evolved in a completely different way. From nearly the beginning, it was rather casually hacked on by huge numbers of volunteers coordinating only through the Internet.

This distributed development model maintains quality through **rapid iteration** and community feedback rather than rigid hierarchical control. Key figures include:

- **Linus Torvalds** -- Principal kernel maintainer and final arbiter
- **Greg Kroah-Hartman** -- Stable branch maintenance lead
- Thousands of individual contributors and corporate sponsors

### Corporate Contributions

Major technology companies contribute significantly to kernel development, including **Intel**, **Red Hat**, **Google**, **Samsung**, **AMD**, and **Microsoft** (which has become one of the largest corporate contributors).

## Government Adoption

Various governments have embraced Linux for sovereignty and cost reasons:

- **India** (Kerala) -- Mandated Linux in school computers
- **France** and **Germany** -- Advanced government adoption initiatives
- **China** -- Uses Linux exclusively for its Loongson processors to achieve technological independence
- **Spain** -- Developed regional distributions (LinEx, Guadalinex) for education

## Impact and Legacy

Linux fundamentally changed how software is developed and distributed. It demonstrated that a large-scale, complex software system could be built by a globally distributed community of volunteers. The collaborative development model pioneered by Linux influenced countless subsequent open-source projects and reshaped the entire technology industry.
